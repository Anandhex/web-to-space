parsePageToIR(html, url, provider, config)

Converts raw HTML into the accessibility-semantic intermediate representation. Returns a flat dictionary of nodes keyed by string id.

const ir = await parsePageToIR(html, url);
console.log(ir.analytics.headingCount);

Parameters

  • html — the raw document source.
  • url — used to resolve relative links.
  • config — toggles for structural inference and AI fallback.

mapIRToScene(ir, config)

Translates each IR role into a typed XR primitive. Every node is mapped; unmapped roles fall through to XRGenericPanel so content is never dropped.

const scene = mapIRToScene(ir, DEFAULT_MAPPER_CONFIG);

computeLayoutPlan(scene, profile, template, overrides)

Places every primitive in three-dimensional space in metres, using a right-handed WebXR coordinate system. Paginates content panels that overflow.

Metres everywhere. Never use pixel units in layout code.
